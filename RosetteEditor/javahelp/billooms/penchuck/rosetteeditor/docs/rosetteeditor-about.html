About Rosette Editor

The rosette editor has a control panel for adjusting the parameters of a simple rosette, and a display panel for viewing the resulting rosette.

By right-clicking in the top of the window, you can un-dock the window and place it anywhere on your screen. You can also choose to maximize, minimize, etc. To restore the original window locations, go to the menu bar under "Window" and select "Reset Window".

Controls:

Amplitude

The amplitude defines the peak-to-peak amplitude of the pattern on the rosette. You can change the amplitude by entering a number directly in the text window, or you can change the position of the slider.

Phase

The phase defines a shift in the rotation of the pattern on the rosette. This program uses the engineering definition of phase. A phase shift of 180 degrees always represents shifting by 1/2 of the pattern regardless of how many repeats are on the rosette. Likewise, a phase shift of 90 degrees always represents shifting by 1/4 of the pattern.

Phase is NOT the absolute rotation of the rosette around the spindle. The benefit of this approach is that you can set the phase and then change the number of repeats while maintaining the same phase offset relative to the pattern.

Repeats

This sets the number of times the pattern is repeated around the rosette.

Style

This sets the basic pattern on the rosette.

Display Window:

The size of the rosette will adapt to the size of the window.
